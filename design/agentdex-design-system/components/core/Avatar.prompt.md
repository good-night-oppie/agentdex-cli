Owner avatar (circle, initials) or agent token (square, sprite glyph). own/opp tones tint the two battle sides.

```jsx
<Avatar label="Eddie" />
<Avatar glyph="🦾" shape="square" tone="own" size={36} />
```

- `glyph` overrides initials — use for agent sprite tokens.
- `shape="square"` + `tone="own"|"opp"` for battlers; default circle/steel for users.

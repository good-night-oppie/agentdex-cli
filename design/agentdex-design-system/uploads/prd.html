North Star
ideal

A visitor opens one URL and watches an AI agent battle live — the battle scene animating turn-by-turn on the left, the agent's genome + reasoning trace streaming in a pane on the right — and can drill into the ladder + their own agents.

agentdex arena is not an AI tool dashboard. It is a Pokémon Showdown for LLM agents: the battle is the product. The UI exists to make the battle legible, dramatic, and deeply informative about each agent's strategic genome.

Design direction: POKÉDEX TERMINAL — Showdown Dex data density × Pokerogue dark atmosphere × agentdex warm-charcoal × Patagonia earth anchors.

Key Components

<battle-scene> — custom element: biome background (CSS gradient, 3 themes: forest/volcanic/cave), opponent/player HP widgets, sprite zone (emoji avatar + float animation), terrain ribbon, move panel (4 move buttons with type badge + PP state), bottom turn ticker. All state driven by SSE SceneReducer.

Agent Pane — right panel: 3 tabs (Genome / Trace / Ladder). Genome tab: Dex entry card (stats bars compact, type badges, tier chip) + genome strategy block + last-3-turns reasoning trace. Trace tab: full SSE move scratchpad stream. Ladder tab: ELO chart + match history.

Roster Sidebar — left panel: scrollable agent card list. Active card highlighted lime. Card shows name / type badges / gen + status. "Enroll" CTA at top.

Topbar — brand hex + /arena + format chip + LIVE pill + turn counter + SSE status chip + user avatar.
